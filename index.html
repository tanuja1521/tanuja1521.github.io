DANCE POINT RECOGNITION

The core idea of the project is based on pose estimation using Posenet repository. The program shows a sample dance video to a user, and asks to copy the performance. The backend deep learning algorithm is coded to compute the variation in the two poses, and normalize the deviations calculated into a percentage score.

Technology Used: TensorFlow,OpenCV,PoseNet

BIONIC LEG

Developed a bionic leg which is an exoskeleton type device for physically impaired people to showcase at the INK Makers Makeathon-2018 at NIT Warangal. It can be improved by using EEG sensors, so that using brain waves frequency of walking 15-40 Hz as signals it can operate automatically.

Technology Used: Arduino UNO, servo motors
